Real Penalty is not installed. Please follow the instructions below to install Real Penalty, or contact your Server Administrator:

  1. Download Real Penalty
  2. Extract AC_Penalty_Tool_ver.zip
  3. Navigate to the Server_Plugin folder inside the extracted folder
  4. Move the tracks and files folders into the plugin folder
  5. Create a new folder called realpenalty (case sensitive) in your Assetto Corsa Server Folder
  6. Copy the contents of the plugin folder into this new realpenalty folder
  7. Make sure that you have set UDP Plugin Address and UDP Plugin Local Port in Server Options
  8. Refresh this page!
